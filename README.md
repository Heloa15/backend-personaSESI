# Backend do Persona SESI 
Programações de backend do Persona SESI dentro do git hub

- > Este repositório contém o desenvolvimento do backend do projeto Persona SESI, responsável por toda a lógica de negócio, gerenciamento de dados e integração entre serviços da aplicação.
  
## Arquitetura
- > O sistema segue boas práticas de arquitetura, como separação de camadas (controllers, services, repositories), visando facilitar manutenção, escalabilidade e testes.

 ## Status do Projeto
- > Em desenvolvimento contínuo, com foco em performance, segurança e expansão de funcionalidades.
